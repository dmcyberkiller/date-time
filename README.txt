## AUTO INSTALL ##

# autoinstall this dati.sh tool for show colorful date and time on your system.

# What is autoinstall.sh and how it's work :-
  autoinstall.sh it's a simple script to install and copy dati.sh tool automatically in your system directory
  and it's automatically add the chmod +x permission on your system.

# How to use this script (autoinstall.sh) :-
  Just type autoinstall.sh in your terminal after cloning it from github and add execute permission, and after install just type dati.sh from anywhere (from any directori's) it's
  run without error.
  
# If I didn't use autoinstall.sh then :-
  After cloning date-time from github, add the permission and copy the dati.sh script and paste on your system directory
  /usr/local/bin folder that's it.
  

*** IF NOT RUNNING ***
If not running just add the execute permission manually.
  
  
## MANUAL INSTALL ##

Clone the date-time from github and you see the dati.sh script. Add the execute permission and just copy the dati.sh script and paste on your system directory usr/local/bin folder, that's it. And you run this script from anywhere just type dati.sh in your terminal.

** IF GET ERROR BY ANY CHANCE **
Add the chmod +x permission after copy this dati.sh script in the destination path.
